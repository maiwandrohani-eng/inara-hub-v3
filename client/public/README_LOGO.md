# INARA Logo

## How to Add Your Logo

1. **Place your INARA logo PNG file here** with the exact filename: `inara-logo.png`

2. **Logo Requirements:**
   - PNG format
   - Transparent background (recommended)
   - High resolution (at least 200x200px for best quality)
   - The colorful interwoven symbol with "inara" text below it

3. **The logo will automatically appear in:**
   - Login page
   - Header navigation
   - Various components throughout the platform

## Current Status

⚠️ **Logo file not found** - The system is currently using a fallback SVG logo.

Once you add `inara-logo.png` to this directory (`client/public/`), it will automatically replace the fallback.

## File Location

The logo file should be placed at:
```
client/public/inara-logo.png
```

After adding the file, refresh your browser to see the new logo.
